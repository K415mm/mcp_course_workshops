# MCP Course Workshops

This repository contains all the interactive Jupyter Notebooks and practical exercises for the Advanced Agentic AI Course.

## 🚀 How to Run the Workshops

These workshops are designed to be run entirely in your web browser using **Google Colab**. You do not need to install Python or any heavy AI libraries on your local machine.

1. Open the `.ipynb` workshop file you want to complete.
2. Click the **"Open in Colab"** button at the top of the notebook.
3. Once in Colab, click **File > Save a copy in Drive** to save your progress.
4. Add your API keys when prompted in the first code cell.
5. Have fun building agents!

## 📂 Repository Structure
- `Workshop_01_CTI_Automation/` - Build an automated Cyber Threat Intelligence pipeline.
- `Workshop_02_Threat_Hunting/` - Use AI to proactively hunt for network anomalies.
- `Workshop_03_Network_Analysis/` - Leverage agents to analyze PCAP files and network flows.
- `Workshop_04_Malware_Analysis/` - Use autonomous agents to perform malware behavior analysis safely.
- `Workshop_05_FastMCP_Deploy/` - Build and deploy Model Context Protocol servers rapidly.
